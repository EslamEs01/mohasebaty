## Summary

Deliver the 6-page data-import workflow (استيراد البيانات) on the approved Mohasebaty design system: `data-import.html` (upload form), `import-mapping.html` (column-mapping table), `import-preview.html` (KPI + preview), `import-history.html` (history log), `import-errors.html` (error detail), and `import-templates.html` (template manager). All pages reuse existing components (zero new CSS goal), are Arabic/RTL, contain only static HTML demo data, and are wired into the shared sidebar and `index.html` as live links across the whole project.

**Important — this scope is already implemented.** Spec `003-data-import-module` produced and committed all six pages, and a static audit (2026-06-11) confirms they conform to the contracts below. Therefore this plan is primarily a **verification + cleanup** plan, not a from-scratch build. The only material code action is reconciling one orphan file (`pages/import-template.html`, singular) left over from an earlier authoring pass. See **Current Implementation State** and **Remaining Work**.
