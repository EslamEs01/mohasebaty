# Implementation Plan: Data Import Module

**Branch**: `003-data-import-module` | **Date**: 2026-06-08 | **Spec**: [spec.md](spec.md)  
**Input**: Feature specification from `specs/003-data-import-module/spec.md`

## Summary

Create 6 complete frontend pages for the data import workflow (استيراد البيانات): a file upload form (`data-import.html`), a column-mapping table (`import-mapping.html`), a data preview (`import-preview.html`), an import history log (`import-history.html`), an error detail view (`import-errors.html`), and a template manager (`import-templates.html`). All pages use the approved Mohasebaty design system — zero new CSS, existing components only. The 6 sidebar coming-soon items for this section are wired as live links across all 23 pages. `index.html` landing cards are converted and counts updated to 23+14.

## Technical Context

**Language/Version**: HTML5 / CSS3 / Tailwind CSS 3.4 (local build) / Vanilla JavaScript ES5 IIFE  
**Primary Dependencies**: Tailwind CSS 3.4 (locally compiled, `npm run build:css`), IBM Plex Sans Arabic (local fonts), `assets/js/main.js` (shared UI interactions — `initSingleSelect`, `initDropzone`, `initOverlays`, `initFilterSelects` already implemented)  
**Storage**: N/A — static prototype, no backend  
**Testing**: Manual visual QA + scriptable static grep audit per `quickstart.md` §3  
**Target Platform**: Browser (Desktop-first, mobile-responsive via existing Tailwind breakpoints)  
**Project Type**: Multi-page static frontend prototype  
**Performance Goals**: Instant page load (static files, no runtime dependencies)  
**Constraints**: No CDN, no backend, no JS-generated content, Arabic RTL, local assets only  
**Scale/Scope**: 6 new pages + sidebar wiring across 23 total pages (17 existing + 6 new)

## Phase Execution Notes

### No New JS Required

All interactive behaviors on the 6 pages are handled by existing `boot()` functions:
- **`.type-grid`/`.type-card` single-select** → `initSingleSelect()` ✅
- **`.seg` button group** → `initSingleSelect()` ✅
- **`.dropzone`** → `initDropzone()` ✅
- **`.filter-select`** → `initFilterSelects()` ✅
- **Add-template modal** → `initOverlays()` ✅ (uses `data-open="#addTemplateModal"` / `data-overlay`)

No additions to `main.js` are needed.

### index.html Landing Cards

The 6 import cards currently appear under the "استيراد البيانات" section header in `index.html`. Each `<div class="landing-card is-soon">` becomes an `<a class="landing-card" href="pages/[filename].html">` with a `.go "فتح الصفحة"` affordance. Same pattern as Spec 002 T019.
