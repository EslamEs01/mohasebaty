# Data Model: Document Categories & Approval Workflow (Spec 009)

Static frontend prototype — all entities are HTML demo data; no enforcement/persistence. VAT is Saudi-first (15% / 0% zero-rated / غير خاضع). Accounts reference the chart of accounts demo names.

---

### Workflow visual (4 steps)
رفع المستند (يصل من العميل) → مراجعة المحاسب (تصنيف وتدقيق) → اعتماد المدير (مراجعة نهائية) → الترحيل للحسابات (تسجيل في الدفاتر).
